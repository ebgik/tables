# tables
Работа с таблицами (Удаление строк, изменение значений в ячейках, добавление строк)

Для работы плагина необходим jquery
Для корректного отображения необходимы bootstrap и font-awesome

Пример использования:

<code>
&lt;table&gt;<br>

&lt;/table&gt;<br>
&lt;script&gt;<br>
    $('.table').editTable();<br>
&lt;/script&gt;
</code>
